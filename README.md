# MANGO: Multi-case Active oNtology-based GO organizer
MANGO is an R package for Gene Ontology (GO) Biological Process enrichment analysis that reduces redundancy in top-ranked results by restructuring enriched terms into ontology-guided term trees based on the GO DAG. It defines and filters active trees using coverage/consistency criteria to suppress structurally driven false positives arising from hierarchical dependencies. MANGO supports single- and multiple- case study designs by integrating enrichment outputs across conditions, providing scoring options and visualization utilities to summarize common and condition-specific biological processes. Optional features include ORA-style soft filtering and fold-change-aware weighting for tree and term prioritization.

>Omics-based functional analysis commonly employs Gene Ontology enrichment to interpret results, mainly focusing on the Biological Process (term) category. However, biologically similar terms often dominate the top ranks, obscuring diverse term signals, and the hierarchical structure of Gene Ontology, together with gene sharing across terms, can induce structural false positives, limiting the interpretability and readability of and confidence in the results. Therefore, we propose MANGO, an analytical framework that reorganizes Gene Ontology outputs into ontology-informed trees and performs structural filtering based on within-tree consistency. In particular, MANGO integrates similar terms into trees to alleviate redundant top-ranked signals and compresses the results into interpretable term units by suppressing isolated, single term–driven signals based on an active-tree criterion. For multiple-case designs, MANGO aligns the results from each comparison into a shared tree structure and systematically classifies common and condition-specific trees using HWES and the relative signal strength across conditions. We performed a parameter search to provide recommended settings specific to the input size and benchmarked MANGO against over-representation analysis and gene set enrichment analysis across 31 single-case analyses, evaluating adjusted p-values, the fraction of terms assigned, and rich factors to assess its interpretation-friendly characteristics. We further demonstrated that MANGO reproduces key term findings reported in diverse public datasets spanning knockout conditions, cancer, differentiation, dose–response, and cohort designs, and we provided R-based visualization functions for both single- and multiple-case settings to facilitate result interpretation. Overall, MANGO mitigates redundancy and structural false positives arising from the Gene Ontology directed acyclic graph structure and offers a practical framework for reproducible term pattern summarization in multicomparison study designs.
